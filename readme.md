# [Easy Blogging Using Python and Pelican](https://github.com/dclong/blog)

## Help Doc 

./main.py -h  

## Search for a Post 

./main.py s pyspark pandas udf

## Edit a Post Returned in the SRPs 

./main.py e 1 

## Add a new Post 

./main.py a this is a new example post 

## Publish Sub Blogs
```
./main.py p -ecmoF
```
