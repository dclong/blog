1. Install [Homebrew](brew.sh).
```sh
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
2. Install XCode from the App store.

3. Install and configure [item2](http://iterm2.com/).

4. Install macvim.
```sh
brew install macvim
```
